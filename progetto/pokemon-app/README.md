# Come installare e avviare l'app

## Requisiti

- Node.js (>= 18)
- npm

## Installazione

1. **Clona la repo:**
   ```bash
   git clone <repo-url>
   cd <nome-cartella-repo>
   ```


# Istruzioni

1. Vai in `progetto/pokemon-app/backend`
2. Esegui `npm install`
